Yahoo CSV Project

October 2020

This program, created using C++, will look for a CSV file and sort it’s data from high to low based on percent change. It’s going to display the number of trading days, open, adj close, and the percent change. The user can choose how many increase record they want to see.


Trigram Project

November 2021

This C++ program will read a text file and look for all possible trigrams. After reading, it will output all the trigrams in alphabetic order and create a CSV file which contains the result.



Persona 5 Tokki

December 2019 - July 2020

This game is started as a fan made, personal project. I wanted to create something meaningful for my first game. So, I combined my friend's story with my favorite game, Persona 5. My friend gave me permission to use this project for any reasons. This game has basic movements, combat system, and puzzle. The AI (Artificial Intelligence) serves it purpose as both enemy and side characters. Animations, cutescenes, and character models were created in Blender and imported to Unreal Engine 4.

For better Quality, watch it on YouTube at https://youtu.be/zKkHn_WC9Cs.

reMatch

December 2020 - in development
